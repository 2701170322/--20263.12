# 公租房管理系统 - 打包部署说明

## 📦 打包方式一：一键启动脚本（推荐）

### 使用方法
1. 双击运行 `start.bat` 文件
2. 等待自动启动前后端服务
3. 浏览器自动访问 http://localhost:3000
4. 按任意键关闭所有服务

### 环境要求
- Python 3.11+
- Node.js 18+
- MySQL 8.0+

---

## 📦 打包方式二：打包为可执行文件

### 后端打包（PyInstaller）

```bash
# 进入后端目录
cd backend

# 安装打包工具
pip install pyinstaller

# 打包为exe
pyinstaller --name "公租房管理系统-后端" \
  --onefile \
  --windowed \
  --add-data "app;app" \
  --add-data "requirements.txt;." \
  run.py
```

打包后会生成 `dist/公租房管理系统-后端.exe`

### 前端打包

```bash
# 进入前端目录
cd frontend

# 安装依赖
npm install

# 打包为静态文件
npm run build

# 打包后的文件在 dist 目录
```

## 🔧 手动启动步骤

### 1. 启动后端
```bash
cd backend
pip install -r requirements.txt
python run.py
```
后端运行在 http://localhost:8001

### 2. 启动前端
```bash
cd frontend
npm install
npm run dev
```
前端运行在 http://localhost:3000

### 3. 访问系统
浏览器打开 http://localhost:3000

## 🗄️ 数据库配置

编辑 `backend/.env` 文件：

```env
DB_HOST=localhost
DB_PORT=3306
DB_USER=root
DB_PASSWORD=你的密码
DB_NAME=public_housing_db
```

## 🐛 常见问题

### 1. 端口被占用
修改 `backend/.env` 中的 PORT 和 `frontend/vite.config.js` 中的端口

### 2. 后端启动失败
检查：
- Python 版本 >= 3.11
- MySQL 服务已启动
- 数据库配置正确

### 3. 前端启动失败
检查：
- Node.js 版本 >= 18
- 运行 `npm install` 安装依赖
